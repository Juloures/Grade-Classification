# Grade Classification

This Python script classifies a student's grade based on the input score.

## Features
- **Input Grade:** The user inputs a grade.
- **Grade Classification:** The script categorizes the grade into one of four classifications:
  - **Excellent:** Grade is 9 or higher.
  - **Good:** Grade is between 7 and 8.9.
  - **Satisfactory:** Grade is between 5 and 6.9.
  - **Insufficient:** Grade is below 5.
- **Display Classification:** The script outputs the corresponding classification.

## How to Run

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/grade-classification.git
    ```
2. Navigate to the project directory:
    ```bash
    cd grade-classification
    ```
3. Run the script:
    ```bash
    python src/grade_classification.py
    ```
